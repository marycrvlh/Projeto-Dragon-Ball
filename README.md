# Dragon Ball Characters 🐉

Projeto desenvolvido durante as aulas do Curso Técnico de Desenvolvimento de Sistemas com foco na criação de uma interface interativa inspirada no universo cyberpunk e em personagens no estilo Dragon Ball.

A aplicação apresenta diferentes personagens fictícios com imagens, descrições e sistema de seleção dinâmica utilizando HTML, CSS e JavaScript.

## 💜 Projeto Online

🔗 Acesse o projeto online:
👉  [projeto-dragon-ball.vercel.app](https://projeto-dragon-ball-three.vercel.app/)


---

## 📋 Funcionalidades

* Seleção interativa de personagens
* Alteração dinâmica de conteúdo
* Layout responsivo
* Imagens adaptadas para desktop e mobile
* Interface inspirada em jogos e anime cyberpunk

---

## 🛠 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

---

## 🎯 Objetivo do Projeto

Este projeto foi criado para praticar:

* Manipulação de elementos com JavaScript;
* Estruturação de páginas com HTML;
* Estilização moderna com CSS;
* Responsividade;
* Interatividade entre elementos da página.
